# 基恩士风格 用户说明书 Word 文档排版规范

此技能定义了工业产品用户说明书的标准排版风格，基于基恩士（KEYENCE）KV-X300 系列用户手册的设计规范。适用于创建专业的产品使用说明书、操作手册、维护指南、培训资料等技术文档。

---

## 一、页面设置

### 1.1 页面尺寸
- **纸张**: B5 (182mm × 257mm) 或自定义尺寸
- **方向**: 纵向
- **特点**: 比 A4 紧凑，便于携带和翻阅

### 1.2 页边距
- **上边距**: 2.0-2.5cm
- **下边距**: 2.0-2.5cm
- **左边距**: 2.5cm (装订侧)
- **右边距**: 2.0cm
- **装订线**: 0.5cm (若需要装订)

### 1.3 页面布局
- **分栏**: 单栏布局（说明书通常单栏）
- **页眉高度**: 1.5cm
- **页脚高度**: 1.5cm
- **正文区域**: 充分利用页面宽度

## 二、配色方案

### 2.1 警告标识系统（核心特征）
说明书最显著的特点是**彩色警告标识系统**：

#### ❌ 危险 (Danger)
- **符号**: 红色背景白色感叹号 ⚠ 或 ❌
- **颜色**: 红色 (RGB 255, 0, 0) 或深红
- **用途**: 可能导致死亡或重伤的情况
- **样式**: 加粗，红色边框，白色背景

#### ⚠ 警告 (Warning)
- **符号**: 橙色/黄色背景黑色感叹号 ⚠
- **颜色**: 橙色 (RGB 255, 165, 0) 或黄色 (RGB 255, 255, 0)
- **用途**: 可能导致重伤或设备损坏的情况
- **样式**: 加粗，橙色边框

#### 💡 注意 (Caution)
- **符号**: 蓝色背景白色信息图标 💡 或 ℹ
- **颜色**: 蓝色 (RGB 0, 112, 192) 或浅蓝
- **用途**: 可能导致轻伤或设备故障的情况
- **样式**: 加粗，蓝色边框

#### ⚡ 备注 (Note)
- **符号**: 绿色背景黑色闪电 ⚡ 或 ◆
- **颜色**: 绿色 (RGB 0, 176, 80) 或深绿
- **用途**: 补充说明、最佳实践
- **样式**: 正常，绿色边框

### 2.2 文字颜色
- **正文**: 黑色 (RGB 0, 0, 0)
- **标题**: 黑色或深灰色
- **警告文字**: 根据警告级别使用对应颜色
- **超链接**: 蓝色 (RGB 0, 112, 192)
- **页眉页脚**: 灰色 (RGB 128, 128, 128)

### 2.3 强调色
- **重要信息**: 蓝色背景或蓝色边框
- **步骤编号**: 蓝色或黑色粗体
- **表格表头**: 浅灰色背景 (RGB 230, 230, 230)

## 三、字体规范

### 3.1 中文字体
- **正文**: 微软雅黑 (Microsoft YaHei) 或 思源黑体 (Source Han Sans CN)
- **标题**: 微软雅黑 粗体
- **警告标识**: 微软雅黑 粗体

### 3.2 英文字体
- **正文**: Arial 或 Helvetica
- **标题**: Arial 粗体
- **代码/参数**: Courier New 或 Consolas
- **产品型号**: Arial 粗体

### 3.3 字号规范
- **封面标题**: 18-22pt，粗体
- **一级标题 (章)**: 16-18pt，粗体
- **二级标题 (节)**: 13-14pt，粗体
- **三级标题**: 11-12pt，粗体
- **正文**: 9-10.5pt
- **表格内容**: 8-9pt
- **脚注/说明**: 8pt
- **页眉页脚**: 8-9pt
- **警告标识**: 10-11pt，粗体

## 四、版面结构

### 4.1 文档结构层次
```
封面页
├── 文档编号 (如 O07CN)
├── 文档类型 (用户手册)
├── 产品系列 (KV-X300 系列)
├── 产品名称和描述
├── 公司 Logo
└── 警告提示："在使用前，请先阅读本手册。请妥善保管，以便随时查阅。"

目录页
├── 目录标题 (目录/目次)
├── 章节列表 (1章, 2章, 3章...)
├── 节列表 (1-1, 1-2, 2-1, 2-2...)
└── 子节列表 (带有层级符号)

内容页 (单栏)
├── 页眉: 文档名称 + 章节号
├── 正文: 章节内容
├── 警告标识: 突出显示
└── 页脚: 页码 + 文档名称

索引/附录
├── 错误代码一览
├── 系统变量一览
└── 其他参考资料
```

### 4.2 页眉设计
- **左侧**: 文档编号或名称 (如 "KV-X300 系列 用户手册")
- **右侧**: 章节号 (如 "第 2 章 技术规格")
- **样式**: 灰色，小字号 (8-9pt)，底部有细线分隔
- **字体**: Arial 或微软雅黑

### 4.3 页脚设计
- **居中**: 页码 (如 "- 3 -" 或 "3/120")
- **样式**: 灰色，小字号，与正文分隔
- **可选**: 文档编号重复显示

### 4.4 章节标题格式

#### 章标题 (一级)
- **编号**: "第 X 章" 或 "X 章" 或 "Xㄖ" (如 "第 1 章 概述")
- **样式**: 16-18pt，粗体，黑色
- **间距**: 
  - 段前: 24-36pt
  - 段后: 12-18pt
- **分页**: 通常新章节从新页开始
- **背景**: 可选浅蓝色或浅灰色背景条

#### 节标题 (二级)
- **编号**: "X-Y" 格式 (如 "1-1 系统配置")
- **样式**: 13-14pt，粗体，黑色
- **间距**:
  - 段前: 18-24pt
  - 段后: 6-12pt

#### 子节标题 (三级)
- **编号**: "① ② ③" 或 "■ □" 或字母 (如 "A B C")
- **样式**: 11-12pt，粗体
- **缩进**: 左缩进 0.5-1.0cm
- **间距**:
  - 段前: 12-18pt
  - 段后: 6pt

## 五、警告标识系统（说明书核心）

### 5.1 警告标识框设计

#### 危险标识框
```
┌─────────────────────────────────────────────────┐
│ ⚠ 危险                                          │ ← 红色背景，白色文字
│                                                  │
│ • 如果不遵守此注意事项，可能导致死亡或重伤。     │ ← 黑色文字
│                                                  │
│ • 具体的危险说明...                             │
│                                                  │
│ • 预防措施...                                   │
└─────────────────────────────────────────────────┘
```

**设计规范**:
- **边框**: 红色粗边框 (2-3pt)
- **标题栏**: 红色背景 (RGB 192, 0, 0)，白色粗体文字
- **内容区**: 白色背景，黑色文字
- **图标**: 红色三角形内白色感叹号
- **圆角**: 可选圆角矩形

#### 警告标识框
```
┌─────────────────────────────────────────────────┐
│ ⚠ 警告                                          │ ← 橙色背景，黑色文字
│                                                  │
│ • 如果不遵守此注意事项，可能导致重伤或设备损坏。 │
│                                                  │
│ • 具体的警告说明...                             │
└─────────────────────────────────────────────────┘
```

**设计规范**:
- **边框**: 橙色粗边框 (2pt)
- **标题栏**: 橙色背景 (RGB 255, 165, 0)，黑色粗体文字
- **内容区**: 白色背景，黑色文字
- **图标**: 橙色三角形内黑色感叹号

#### 注意标识框
```
┌─────────────────────────────────────────────────┐
│ 💡 注意                                          │ ← 蓝色背景，白色文字
│                                                  │
│ • 如果不遵守此注意事项，可能导致轻伤或设备故障。 │
│                                                  │
│ • 具体的注意事项...                             │
└─────────────────────────────────────────────────┘
```

**设计规范**:
- **边框**: 蓝色边框 (1-2pt)
- **标题栏**: 蓝色背景 (RGB 0, 112, 192)，白色粗体文字
- **内容区**: 白色背景，黑色文字
- **图标**: 蓝色圆形内白色 "i" 信息图标

#### 备注标识框
```
┌─────────────────────────────────────────────────┐
│ ⚡ 备注                                          │ ← 绿色背景，黑色文字
│                                                  │
│ • 补充说明、最佳实践、常见问题等。             │
│                                                  │
│ • 额外的提示...                                 │
└─────────────────────────────────────────────────┘
```

**设计规范**:
- **边框**: 绿色边框 (1pt)
- **标题栏**: 绿色背景 (RGB 0, 176, 80)，黑色粗体文字
- **内容区**: 白色背景，黑色文字
- **图标**: 绿色圆形内黑色闪电或对勾

### 5.2 警告标识的使用原则
1. **层级清晰**: 危险 > 警告 > 注意 > 备注
2. **位置醒目**: 在相关操作步骤之前
3. **内容具体**: 明确说明危险后果和预防措施
4. **语言简洁**: 使用简短、直接的语句
5. **重复强调**: 关键警告可在多处重复

### 5.3 警告标识的排版
- **宽度**: 与正文等宽或略窄
- **位置**: 正文内容之前或之中
- **间距**: 前后留有足够空白 (12-18pt)
- **编号**: 可选编号 (如 警告1, 警告2)

## 六、正文排版

### 6.1 正文段落
- **行距**: 1.2-1.3 倍行距
- **段间距**: 6-12pt
- **首行缩进**: 无（说明书通常无首行缩进）
- **对齐**: 两端对齐 (Justify)
- **字体**: 9-10.5pt

### 6.2 操作步骤格式

#### 编号步骤
```
1. 第一步操作说明
   - 详细说明...
   - 注意事项...

2. 第二步操作说明
   - 详细说明...

3. 第三步操作说明
   - 详细说明...
```

**设计规范**:
- **编号**: 阿拉伯数字加粗 (1. 2. 3.)
- **缩进**: 步骤内容左缩进 0.5-1.0cm
- **行距**: 1.15-1.25 倍
- **间距**: 步骤之间 6-12pt

#### 子步骤
- **符号**: "•" 或 "-" 或 "▸"
- **缩进**: 比主步骤多缩进 0.5cm
- **样式**: 正常，非粗体

### 6.3 列表格式

#### 无序列表
- **符号**: • (实心圆点)、- (短横线)、▸ (三角箭头)
- **缩进**: 左缩进 0.5cm
- **项目间距**: 3-6pt
- **样式**: 正常，非粗体

#### 有序列表
- **格式**: (1) (2) (3) 或 1. 2. 3. 或 ① ② ③
- **缩进**: 左缩进 0.5cm
- **样式**: 编号加粗

### 6.4 参数规格格式
```
测量范围: 0 to 100 kPa
精度: ±0.04% of Span
响应时间: 250 ms
电源: 24 V DC
```

**设计规范**:
- **标签**: 加粗，后跟冒号
- **值**: 正常，右对齐或对齐到统一位置
- **单位**: 斜体或正常，与数值之间有空格
- **对齐**: 可选表格形式对齐

## 七、表格规范

### 7.1 表格样式
- **表头背景**: 浅灰色 (RGB 230, 230, 230) 或浅蓝色
- **表头文字**: 黑色粗体，9pt
- **表格内容**: 黑色，8-9pt
- **边框**: 细实线 (0.5pt)，黑色或深灰色
- **单元格内边距**: 上下 3-5pt，左右 5-8pt
- **表格宽度**: 与正文等宽

### 7.2 表格对齐
- **表头**: 居中对齐
- **数字列**: 右对齐或小数点对齐
- **文字列**: 左对齐
- **单位列**: 居中或右对齐

### 7.3 表格标题
- **位置**: 表格上方，左对齐
- **格式**: "表 X-X XXXX" 或 "Table X-X XXXX"
- **样式**: 10-11pt，粗体
- **编号**: 按章节编号 (如 表 2-1, 表 3-2)

### 7.4 表格示例结构
```
表 2-1 CPU单元规格
┌──────────────┬──────────────┬──────────────┐
```

### 7.5 交替行颜色
- **奇数行**: 白色背景
- **偶数行**: 浅灰色背景 (RGB 245, 245, 245)
- **用途**: 提高长表格的可读性

## 八、图片与图示

### 8.1 产品图片
- **位置**: 内容页中，与相关文字相邻
- **尺寸**: 适中，不影响文字阅读
- **边框**: 细边框或无边框
- **环绕**: 上下型环绕
- **标题**: 图片下方，居中

### 8.2 图示规范
- **标题格式**: "图 X-X XXXX" 或 "Figure X-X XXXX"
- **位置**: 图片下方，居中
- **样式**: 9-10pt，正常
- **编号**: 按章节编号 (如 图 1-1, 图 3-2)

### 8.3 接线图/示意图
- **线条**: 统一宽度 (0.5-1pt)，黑色或深色
- **标注**: 清晰的文字标注，指向具体部件
- **颜色**: 黑白为主，彩色用于强调关键部分
- **分辨率**: 至少 300 DPI

### 8.4 操作截图/界面图
- **边框**: 细边框，区分图片和背景
- **标注**: 使用箭头或圆圈标注关键按钮/区域
- **说明**: 图片下方或旁边有文字说明

### 12.3 参数说明格式
```
参数名称: D100

范围: 0 to 65535
默认值: 0
说明: 设定目标值
```

### 12.4 单位规范
- **格式**: 数值 + 空格 + 单位
- **示例**: 10 kPa, 3.8 mA, 250 ms, 24 V DC
- **负值**: 前缀负号，如 -100 kPa

## 十七、注意事项

### 17.1 语言风格
1. **简洁明了**: 使用简短、直接的语句，避免复杂句式
2. **主动语态**: 使用主动语态，明确动作主体
3. **一致性**: 术语、格式、风格保持一致
4. **用户导向**: 从用户角度出发，解决实际问题

### 17.2 安全性
1. **警告前置**: 重要警告放在操作步骤之前
2. **后果说明**: 明确说明不遵守警告的后果
3. **预防措施**: 提供具体的预防措施和解决方案
4. **重复强调**: 关键安全信息在多处重复

### 17.3 可用性
1. **易于查找**: 清晰的目录、索引、页码
2. **易于理解**: 图示、示例、步骤分解
3. **易于操作**: 步骤编号、参数表、检查清单
4. **易于维护**: 版本控制、修订历史
