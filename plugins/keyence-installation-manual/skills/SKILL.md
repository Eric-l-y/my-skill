# 基恩士风格 安装手册 Word 文档排版规范

此技能定义了工业产品安装手册的标准排版风格，基于基恩士（KEYENCE）VS系列 EtherNet/IP 安装手册的设计规范。适用于创建设备连接指南、系统配置手册、通讯协议设置手册、PLC连接指南等安装类技术文档。

---

## 一、页面设置

### 1.1 页面尺寸
- **纸张**: A4 (210mm × 297mm)
- **方向**: 纵向
- **特点**: 标准A4尺寸，便于打印和存档

### 1.2 页边距
- **上边距**: 2.0-2.5cm
- **下边距**: 2.0-2.5cm
- **左边距**: 2.5cm (装订侧)
- **右边距**: 2.0cm
- **装订线**: 0.5cm (若需要装订)

### 1.3 页面布局
- **分栏**: 单栏布局
- **页眉高度**: 1.5cm
- **页脚高度**: 1.5cm
- **正文区域**: 充分利用页面宽度

## 二、文档结构

### 2.1 整体结构
```
封面页 (Page 1)
├── 版权声明 (右上角)
├── 篇名 (如 "EtherNet/IP篇")
├── 系列名 (如 "（Allen-Bradley ControlLogix系列）")
├── 文档编号 (如 "I52CN")
└── 公司名称 (如 "株式会社基恩士 图像系统事业部")

目录页 (Page 2-3)
├── 标题: "目录"
├── 章节列表 (1. 2. 3. ...)
├── 小节列表 (1.1 1.2 1.3 ...)
└── 页码对齐

正文页 (Page 4+)
├── 页眉: 篇名 + 系列名 | 公司名称
├── 章标题: "1. 章节名称"
├── 节标题: "1.1 节名称"
├── 子节标题: "1.1.1 子节名称"
├── 操作步骤: ①②③④⑤⑥⑦⑧⑨⑩

### 2.2 页眉设计
- **左侧**: 篇名 + 系列名 (如 "EtherNet/IP篇　Allen-Bradley ControlLogix系列")
- **右侧**: 公司名称 (如 "株式会社基恩士　图像系统事业部")
- **样式**: 黑色，小字号 (8-9pt)
- **分隔**: 左右内容之间用全角空格或制表符分隔

### 2.3 页脚设计
- **居中**: 页码 (如 "4", "5", "6")
- **样式**: 黑色，小字号 (8-9pt)
- **格式**: 纯数字，无前后缀

## 三、标题层级系统

### 3.2 节标题 (二级)
- **格式**: "1.1 节名称" (章号 + 点 + 节号 + 空格)

- **样式**: 13-14pt，粗体，黑色

- **位置**: 左对齐

### 3.3 子节标题 (三级)
- **格式**: "1.2.1 子节名称" (章号 + 点 + 节号 + 点 + 子节号 + 空格)
- **示例**: "1.2.1 下载VS Creator", "1.2.2 安装VS Creator"
- **样式**: 11-12pt，粗体，黑色
- **间距**: 段前 12-18pt，段后 6pt
- **位置**: 左对齐

## 四、操作步骤系统

### 4.1 圆圈数字步骤
安装手册最显著的特点是使用**圆圈数字**标注操作步骤：

#### 基本格式
```
① 第一步操作说明
② 第二步操作说明
③ 第三步操作说明
④ 第四步操作说明
⑤ 第五步操作说明
⑥ 第六步操作说明
⑦ 第七步操作说明
⑧ 第八步操作说明
⑨ 第九步操作说明
⑩ 第十步操作说明
```

#### 设计规范
- **符号**: ①②③④⑤⑥⑦⑧⑨⑩ (Unicode 圆圈数字)
- **字体**: 与正文字体相同
- **字号**: 10-11pt
- **样式**: 正常或粗体
- **颜色**: 黑色
- **缩进**: 左缩进 0.5-1.0cm
- **间距**: 步骤之间 6-12pt

### 4.2 步骤内容格式
```
①启动软件，选择"新建项目"。
②在对话框中输入以下信息：
  - 项目名称: TestProject
  - 设备型号: VS-8000
  - IP地址: 192.168.0.10
③单击"确定"完成创建。
```

**设计规范**:
- **步骤编号**: 圆圈数字，左对齐
- **步骤内容**: 编号后直接跟文字，无额外标点
- **子项**: 使用短横线 (-) 或圆点 (•) 缩进列出
- **对齐**: 两端对齐或左对齐

### 4.3 子步骤格式
```
①启动软件后，执行以下操作：
  (1) 选择"文件"菜单
  (2) 单击"新建"
  (3) 输入项目名称
②确认设置后，单击"确定"。
```

**设计规范**:
- **子编号**: (1) (2) (3) 或 a) b) c)
- **缩进**: 比主步骤多缩进 0.5-1.0cm
- **样式**: 正常，非粗体

### 5.1 表格样式
- **表头背景**: 浅灰色 (RGB 230, 230, 230) 或无背景
- **表头文字**: 黑色粗体，9-10pt
- **表格内容**: 黑色，9-10pt
- **边框**: 细实线 (0.5pt)，黑色或深灰色
- **单元格内边距**: 上下 3-5pt，左右 5-8pt
- **表格宽度**: 与正文等宽

### 5.2 表格对齐
- **表头**: 居中对齐
- **文字列**: 左对齐
- **数值列**: 右对齐或居中
- **代码/地址**: 使用等宽字体 (Courier New)

```
数据类型          VS系列字节地址    PLC的Name

**设计规范**:
- **列宽**: 根据内容自适应

## 六、提示与注释系统

### 6.1 重要提示 (重要)
- **标记**: "重要" 或 "★"
- **样式**: 粗体，红色 (RGB 192, 0, 0) 或黑色
- **位置**: 相关内容之前或之后
- **边框**: 可选红色左边框或红色背景

**示例**:
```
重要
在PLC侧，为了确认VS相机将Command Complete设为ON，
需要在Command Complete变为ON之前、也就是上升沿之前，让Command Request保持为ON。
```

### 6.2 参考注释 (※)
- **标记**: "※" (日文参考符号)
- **样式**: 正常，非粗体
- **位置**: 相关内容之后
- **用途**: 补充说明、注意事项、替代方法

**示例**:
```
※关于"Style"，只要没有指定，就是"Decimal"。
※使用PoE交换集线器（CG-SW100）时，使用以下的M12X-M12X的Ethernet连接线。
```

### 6.3 脚注标记 (★1)
- **标记**: ★1, ★2, ★3 等
- **样式**: 上标或正常
- **位置**: 文内标记，解释在页面底部或附近
- **用途**: 对复杂流程的补充说明

## 七、时序图与流程说明

### 7.1 时序图文字描述
安装手册中常用文字描述时序图：

```
①PLC确认Trigger Ready为ON、Trigger Ack为OFF后，将Trigger设为ON。
②VS系列接受触发输入后，开始拍摄，将Trigger Ready设为OFF，Trigger Ack设为ON。
③拍摄完成、可以开始下一次拍摄的VS系列，将Trigger Ready设为ON。
④PLC确认TriggerAck已变为ON后，将Trigger设为OFF。
⑤与④联动，VS系列将Trigger Ack设为OFF。
```

**设计规范**:
- **编号**: 使用圆圈数字 ①②③④⑤
- **对齐**: 左对齐
- **间距**: 步骤之间适当间距
- **关键词**: ON/OFF、上升沿/下降沿 使用粗体或等宽字体

### 7.2 流程说明
```
VS系列 → PLC:
  将Result Ready设为ON
  向Output Data中写入输出数据
  写入完成后，将Result Ready设为ON

PLC → VS系列:
  通过Result Ready为ON确认数据写入完成
  将Result Ack设为ON
  检测到Result Ready已变成OFF
  将Result Ack设为OFF
```
