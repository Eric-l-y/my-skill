# 横河电机风格 Word 文档排版规范

此技能定义了工业产品技术规格书的标准排版风格，基于横河电机（Yokogawa）产品规格书的设计规范。适用于创建专业的工业产品技术文档、规格书、产品手册等。

---

## 一、页面设置

### 1.1 页面尺寸
- **纸张**: A4 (210mm × 297mm)
- **方向**: 纵向

### 1.2 页边距
- **上边距**: 2.0cm
- **下边距**: 2.0cm
- **左边距**: 2.0cm
- **右边边距**: 2.0cm
- **装订线**: 0.5cm (若需要装订)

### 1.3 分栏
- **正文区域**: 双栏布局
- **栏宽**: 等宽 (每栏约 8.5cm)
- **栏间距**: 0.5cm

---

## 二、配色方案

### 2.1 主色调
- **公司蓝**: RGB(0, 84, 168) 或 CMYK(100, 50, 0, 34) — 用于标题、logo、强调文字
- **标准黑**: RGB(0, 0, 0) — 用于正文
- **灰色背景**: RGB(230, 230, 230) — 用于表格表头
- **浅灰**: RGB(245, 245, 245) — 用于表格交替行

### 2.2 文字颜色
- **正文**: 黑色
- **标题**: 深蓝或黑色
- **超链接/导航**: 蓝色
- **页脚/页眉**: 灰色或浅黑色

---

## 三、字体规范

### 3.1 中文字体
- **正文**: 微软雅黑 (Microsoft YaHei) 或 思源黑体 (Source Han Sans CN)
- **标题**: 微软雅黑 粗体
- **数字/英文**: Arial 或 Times New Roman

### 3.2 英文字体
- **正文**: Arial 或 Helvetica
- **标题**: Arial 粗体
- **代码/技术规格**: Courier New 或 Consolas

### 3.3 字号规范
- **文档标题**: 18-20pt，粗体
- **一级标题 (■)**: 14-16pt，粗体
- **二级标题 (□)**: 12-14pt，粗体
- **三级标题**: 11-12pt，粗体
- **正文**: 9.5-10.5pt
- **表格内容**: 8-9pt
- **页眉/页脚**: 8-9pt
- **注释/说明**: 8pt，斜体

---

## 四、版面结构

### 4.1 文档结构层次
```
封面页
├── 文档标题 (粗体，大号)
├── 产品型号/名称
├── 公司 Logo 和名称
├── 文档编号和版本
└── 日期

目录/索引页 (可选)
├── 目录标题
└── 目录条目

内容页 (双栏)
├── 页眉: <<Contents>> <<Index>> (导航链接)
├── 栏1: 主要内容
├── 栏2: 补充内容或表格
└── 页脚: 文档号 + 版权信息 + 页码
```

### 4.2 页眉设计
- **位置**: 页面顶部
- **内容**: "<<Contents>> <<Index>>" (可点击的导航链接)
- **样式**: 蓝色，小字号，居中或右对齐
- **格式**: 带下划线或超链接样式

### 4.3 页脚设计
- **位置**: 页面底部
- **内容格式**: 
  - 文档编号 (如 GS 01C25R04-01EN)
  - 版权声明 (©Copyright 年份, 公司名称)
  - 版本号和日期 (如 32nd Edition Jun. 2026)
  - 页码 (右下角或居中)
- **样式**: 灰色，小字号，与正文分隔

---

## 五、标题与段落

### 5.1 标题层级

#### 一级标题 (■ 标记)
- **符号**: ■ (实心方块) + 标题文字
- **样式**: 粗体，14-16pt，深蓝色或黑色
- **间距**: 
  - 段前: 12pt
  - 段后: 6pt
- **分栏**: 跨越双栏，通栏显示

#### 二级标题 (□ 标记)
- **符号**: □ (空心方块) + 标题文字
- **样式**: 粗体，12-14pt，黑色
- **间距**:
  - 段前: 8pt
  - 段后: 4pt

#### 三级标题 (缩进)
- **样式**: 粗体，11-12pt，黑色
- **缩进**: 左缩进 0.5cm
- **间距**:
  - 段前: 6pt
  - 段后: 3pt

### 5.2 正文段落
- **行距**: 1.15-1.25 倍行距
- **段间距**: 0-6pt
- **首行缩进**: 无
- **对齐**: 两端对齐 (Justify)
- **栏内换行**: 保持段落完整性，避免孤行

---

## 六、表格规范

### 6.1 表格样式
- **表头背景**: 浅灰色 (RGB 230, 230, 230)
- **表头文字**: 黑色粗体，9pt
- **表格内容**: 黑色，8-9pt
- **边框**: 细实线 (0.5pt)，黑色或深灰色
- **单元格内边距**: 上下 3pt，左右 5pt

### 6.2 表格对齐
- **表头**: 居中对齐
- **数字列**: 右对齐或小数点对齐
- **文字列**: 左对齐
- **单位列**: 居中或右对齐

### 6.3 复杂表格处理
- **合并单元格**: 用于分类标题行
- **跨列**: 用于表格标题
- **嵌套表格**: 用于复杂规格对比
- **脚注**: 表格下方用小字号标注

### 6.4 表格示例结构
```
┌──────────────┬──────────────┬──────────────┐
│  参数名称     │    单位      │   范围值      │  ← 灰色背景，粗体
├──────────────┼──────────────┼──────────────┤
│  测量范围     │    kPa       │   1 to 100   │  ← 正常行
├──────────────┼──────────────┼──────────────┤
│  精度等级     │    %         │   ±0.04      │  ← 正常行
└──────────────┴──────────────┴──────────────┘
```

---

## 七、列表与要点

### 7.1 无序列表
- **符号**: • (实心圆点) 或 ◦ (空心圆点)
- **缩进**: 左缩进 0.5cm
- **项目间距**: 3-4pt
- **子项缩进**: 额外 0.5cm

### 7.2 有序列表
- **格式**: (1), (2), (3) 或 1., 2., 3.
- **缩进**: 与无序列表一致

### 7.3 列表样式示例
```
• 主要特性1
  子特性说明
• 主要特性2
• 主要特性3
  (1) 子项1
  (2) 子项2
```

---

## 八、图片与图表

### 8.1 产品图片
- **位置**: 封面页右侧或内容页指定位置
- **尺寸**: 适中，不影响文字阅读
- **边框**: 无边框或细边框
- **环绕**: 上下型或嵌入型

### 8.2 图表规范
- **标题**: 图表下方，居中
- **编号**: 按章节编号 (如 图1-1, 表2-3)
- **来源**: 图表下方注明来源 (可选)
- **分辨率**: 至少 300 DPI (印刷)

### 8.3 技术图示
- **线宽**: 统一，0.5-1pt
- **颜色**: 黑白或深色为主，彩色用于强调
- **标注**: 清晰的尺寸线和标签

## 九、特殊元素

### 9.1 脚注与尾注
- **脚注**: 页面底部，细线分隔
- **尾注**: 文档末尾
- **标记**: *, **, †, ‡, §
- **样式**: 8pt，斜体

### 9.2 注释与说明
- **样式**: "Note:" 开头，斜体
- **位置**: 相关内容下方
- **缩进**: 左缩进 0.5cm

### 9.3 警告与注意
- **标记**: ▲ 或 ⚠
- **样式**: 粗体或彩色背景
- **位置**: 突出显示

### 9.4 公式与数学表达
- **样式**: 斜体变量，正体单位
- **对齐**: 居中或右对齐编号
- **编号**: 括号内，右对齐

### 10.1 布局
- **上部**: 公司 Logo (右上角或左上角)
- **中部**: 
  - 文档类型 (如 "General Specifications")
  - 产品型号 (如 "EJX930A")
  - 产品名称 (如 "Multivariable Transmitter")
- **下部**:
  - 文档编号 (如 GS 01C25R04-01EN)
  - 版本信息 (如 [Style: S1])

## 十一、版权与法律信息

### 11.1 版权声明格式
```
©Copyright [年份], [公司全称]
All Rights Reserved.
```

### 11.2 文档编号格式
- **格式**: [类型代码][版本]-[序号][语言代码]
- **示例**: GS 01C25R04-01EN
- **位置**: 页眉、页脚、封面

### 11.3 版本控制
- **版本号**: 数字或字母
- **日期**: 月. 日, 年
- **修订说明**: 可选

---

## 十二、代码与技术规格

### 12.1 代码块
- **字体**: Courier New 或 Consolas
- **背景**: 浅灰色
- **边框**: 细实线
- **行号**: 可选

### 12.2 技术参数
- **格式**: 参数名 = 值 单位
- **示例**: Span = 1 to 100 kPa
- **精度**: 保持一致性 (小数位数统一)

### 12.3 单位规范
- **格式**: 数值 + 空格 + 单位
- **示例**: 10 kPa, 3.8 mA, 250 ms
- **负值**: 前缀负号，如 -100 kPa

---

## 十三、多语言支持

### 13.1 中英文混排
- **中文**: 使用中文字体
- **英文/数字**: 使用英文字体
- **标点**: 中文标点用全角，英文标点用半角

### 13.2 国际化
- **日期格式**: Jun. 2026 或 2026年6月
- **数字格式**: 千位分隔符 (1,000) 或空格 (1 000)
- **度量单位**: 公制为主，可选英制

---

## 十四、印刷与输出

### 14.1 分辨率
- **文字**: 矢量，无限分辨率
- **图片**: 300 DPI 以上
- **线条**: 600 DPI 以上

### 14.2 颜色模式
- **印刷**: CMYK
- **屏幕**: RGB
- **黑白打印**: 确保灰度可读

### 14.3 PDF 输出
- **标准**: PDF/X-1:2001 (印刷) 或 PDF/A (存档)
- **字体**: 嵌入所有字体
- **压缩**: 适当压缩图片

## 十六、应用示例

### 16.1 产品规格书
适用场景：工业设备、仪器仪表、电子产品规格书
关键特征：详细的技术参数表格，多层级标题，专业术语

适用场景：安装指南、操作手册、维护手册

## 注意事项

1. **保持专业性**: 使用正式、客观的语言，避免口语化表达
2. **一致性**: 整个文档保持统一的格式和风格
3. **可读性**: 合理的行距、段距、字体大小，确保易读
4. **准确性**: 技术参数、单位、公式必须准确无误
5. **完整性**: 包含所有必要的信息，如版本、日期、联系人
6. **可访问性**: 考虑不同用户的阅读需求，如字体大小可调
