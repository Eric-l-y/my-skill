# My Claude Skills

我的 Claude Code 个人技能库，包含日常开发中积累的经验和最佳实践。

## 安装

```bash
claude plugin install github.com/Eric-l-y/my-skill
```

## 技能列表

| 技能 | 说明 | 调用方式 |
|------|------|----------|
| `/code-review-checklist` | 代码审查检查清单 | 用户手动调用 |
| `/git-commit-style` | Git 提交信息规范 | 用户手动调用 |

## 更新

```bash
claude plugin update my-claude-skills
```
